# Google Codelab - Gen AI for Python Developers
[Google Developers Codelabs](https://codelabs.developers.google.com/)

[Google Developers - Codelab - Automatically Deploy Generative AI Python Web Application from Version Control to Cloud Run](https://codelabs.developers.google.com/codelabs/deploy-from-github/gen-ai-python)

In this lab, you write a web application and configure Cloud Run to automatically deploy your application when a change is made to the source code of your application. Then you modify your application and deploy it again.

This [website](https://codelab-genai-python-412316639419.us-central1.run.app/) shows 10 interesting facts about an herb. Eg: [oregano](https://codelab-genai-python-412316639419.us-central1.run.app/?herb=oregano)
